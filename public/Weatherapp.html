Weather App

The project is a Tkinter-based weather app that fetches and displays real-time weather data for a specified city from the OpenWeatherMap API.

Python
Tkinter
Requests library

Overview

This project involves creating a weather app using the Tkinter GUI library in Python. Users can input a city name, and the app fetches real-time weather data from the OpenWeatherMap API. The app then displays key information like temperature, humidity, and wind speed in a user-friendly interface.
